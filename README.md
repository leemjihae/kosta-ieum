# 건강이음 - 심리상담 예약·기록 통합 관리 플랫폼

------------
## 프로젝트 소개

<img width="200" alt="image" src="https://github.com/user-attachments/assets/cd878bdb-44e2-4494-9359-9b6f0b1768ea" />


- **기간**: 2025.08.12 ~ 2025.10.17 (2달)

- **형태**: JSP 기반 MVC 패턴과 MyBatis를 활용한 웹 애플리케이션
  
- **주제**: 정신과 진료에 대한 심리적 문턱을 낮추기 위해 온라인 예약 및 상담 관리 서비스를 제공하고, 기록과 커뮤니티 기능을 통해 지속적인 정신건강 관리를 지원하는 공익적 플랫폼을 구축한다.
  
- **팀 구성**: 김종조, 박은솔, 임지혜, 최지성 (4인)

- **맡은 역할**: 병원 조회, 상담 예약, 자가검진 서비스
